# Wildlife Protection AI Project — Portfolio Summary

Developed a Streamlit-based AI portfolio prototype to demonstrate how forest audio recordings could be analysed for suspicious activity detection, including vehicle-like sounds, chainsaw-like sounds and unusual human activity.

## Key Contributions

- Designed an interview-ready Streamlit interface with built-in demo scenarios.
- Created a clear workflow for audio input, feature extraction, classification and risk-level output.
- Included active learning logic to prioritise uncertain samples for human review.
- Structured the project for GitHub and Streamlit deployment.

## Tools

Python, Streamlit, NumPy, Pandas, Scikit-learn workflow design, audio feature extraction concept, active learning.
